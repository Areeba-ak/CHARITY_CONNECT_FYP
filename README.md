# Charity Connect (AI-Powered Donation Platform)
Charity Connect is a full-stack web application designed to connect donors with individuals in need. It provides a secure, transparent, and user-friendly platform for making donations, managing charity requests, and tracking transactions using blockchain technology and AI-powered support.


**Project Highlights**
- Secure & transparent donation tracking using blockchain
- AI-powered chatbot for donor support
- Role-based dashboards (Admin, Donor, Needy)
- Real-time donation tracking & reporting
- Modern responsive UI with smooth UX
  
**Key Features** 

**Admin**
- Manage users (Donors & Needy)  
- Approve and verify charity stories  
- Monitor donations and generate reports  

**Donor**  
- Browse verified charity stories  
- Make secure donations (Stripe / PayPal / Credit Card)  
- Track donation history  
- AI-powered support chatbot  

**Needy** 
- Submit stories for assistance  
- Track funding progress  
- Receive donation updates  

**Blockchain**
- Immutable record of donation transactions  
- Transparent and tamper-proof tracking  

**Tech Stack**  
- **Frontend:** React.js, Material UI  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Blockchain:** Solidity, Hardhat  
- **Authentication:** JWT, Passport.js  
- **Payments:** Stripe, PayPal, Credit Card  
- **Tools:** Git, GitHub, VS Code  


**Clone repository**

- git clone https://github.com/Areeba-ak/CHARITY_CONNECT_FYP 
- cd CHARITY_CONNECT_FYP


**Install backend dependencies**
- cd Backend
- npm install


**Install frontend dependencies**
- cd ../client
- npm install


**Start backend server**
- cd ../Backend
- npm start


**Start frontend app**
- cd ../client
- npm start
